# Phase 7: Order Intake + POS + Retail Counter

**Document Version:** 1.0  
**Date:** January 17, 2026  
**Status:** UX Design Specification

---

## 1. EXECUTIVE SUMMARY

This document defines the fast intake layer for a metals/plastics service center, covering three distinct but integrated workflows:

1. **Quote-to-Order** - Formal quotation process for larger/complex jobs
2. **Direct Order Entry** - Standard orders with known specs
3. **Walk-in POS / Retail Counter** - Fast transactions for will-call and counter sales

**Design Goals:**
- **Speed:** Counter sale in < 60 seconds
- **Accuracy:** Zero tolerance for material/spec errors
- **Flexibility:** Handle everything from $50 bar stock to $50,000 custom orders
- **Intelligence:** Smart defaults, history-based suggestions, auto-routing

---

## 2. ORDER TYPES & WORKFLOWS

### 2.1 Order Type Matrix

| Order Type | Entry Time Target | Typical Value | Processing | Payment |
|------------|-------------------|---------------|------------|---------|
| **Counter Sale** | < 60 sec | $50 - $500 | None or simple cut | Immediate |
| **Will-Call** | < 2 min | $100 - $2,000 | Pre-picked, maybe cut | On pickup |
| **Quick Order** | < 5 min | $500 - $5,000 | Standard processing | Net terms |
| **Standard Order** | 5-15 min | $1,000 - $25,000 | Multi-step processing | Net terms |
| **Quote-to-Order** | Convert existing | Any | Complex/custom | Per quote terms |
| **Blanket Release** | < 2 min | Varies | Per release schedule | Net terms |

## 14. IMPLEMENTATION ROADMAP

### Phase 1: Core POS (Weeks 1-4)
- Basic counter sale flow
- Cash/card payment processing
- Product search and add
- Receipt printing
- Operator login

### Phase 2: Order Management (Weeks 5-8)
- Quote creation
- Quote-to-order conversion
- Work order generation
- Customer lookup and management
- Credit/account payment

### Phase 3: Processing Integration (Weeks 9-12)
- Cut-to-length orders
- Routing template application
- Shop floor queue integration
- Remnant tracking

### Phase 4: Advanced Features (Weeks 13-16)
- Division-aware materials
- Supply bundling and suggestions
- Multi-location inventory
- Advanced reporting
