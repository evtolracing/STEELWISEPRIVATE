# 17 - UI COMPONENT LIBRARY

## Design System

### Colors
```
Primary:    #1E3A5F (Steel Blue)
Secondary:  #FF6B35 (Industrial Orange)
Success:    #28A745
Warning:    #FFC107
Danger:     #DC3545
Neutral:    #6C757D
Background: #F8F9FA
Surface:    #FFFFFF
```

### Typography
```
Headers:    Inter Bold (24/20/16px)
Body:       Inter Regular (14px)
Data:       JetBrains Mono (13px) - for numbers/codes
```

### Core Components

| Component | Usage |
|-----------|-------|
| DataTable | Sortable, filterable data grids |
| StatusBadge | Colored status indicators |
| MetricCard | KPI display boxes |
| SearchInput | Global and inline search |
| ActionButton | Primary/secondary actions |
| FormField | Input with label and validation |
| Modal | Dialogs and confirmations |
| TabPanel | Content organization |
| Timeline | Event history display |
| TreeView | Hierarchical data (genealogy) |

### Status Indicators
```
● Green  = Active / Available / Pass
◐ Yellow = Partial / Warning / Pending
○ Gray   = Inactive / Empty
⊘ Red    = Hold / Blocked / Fail
```

### Responsive Breakpoints
```
Mobile:   < 768px
Tablet:   768px - 1024px
Desktop:  > 1024px
```

### Accessibility
- WCAG 2.1 AA compliance
- Keyboard navigation
- Screen reader support
- High contrast mode
